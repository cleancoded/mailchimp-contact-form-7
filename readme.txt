== Description ==

Integrate Contact Form 7 with [MailChimp](https://github.com/cleancoded/mailchimp-contact-form-7). Automatically add form submissions to predetermined lists in MailChimp, using its latest API. This MailChimp Extension supports multiple mailing lists and API Keys.

= Key Features =

* Easy to use
* Use a different mailing list per contact form
* Single opt-in - no confirmation email sent to subscribers
* Double opt-in - confirmation email send to subscribers
* Latest MailChimp API
* Regularly updated

= Please note*** =

* Support for groups - coming soon
* Support for segments - coming soon

= Requirements =

1. Self hosted WordPress.org installation (4.0 or higher)
2. Contact Form 7 (4.2 or higher)
3. MailChimp account

== Installation ==

1. Upload the entire ‘[mailchimp-contact-form-7](https://github.com/cleancoded/mailchimp-contact-form-7)' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find a ‘MailChimp: Subscriber List Details' menu in your Contact Form 7 editor.

For basic usage, you can also have a look at the [MailChimp integration for Contact Form 7 homepage](http://github.com/cleancoded/mailchimp-contact-form-7), you can also have a look at the [Contact Form 7 homepage](http://contactform7.com).
